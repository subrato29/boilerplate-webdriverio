This is a WebDriverIO framework to test web application

boilerplate-webdriverio

Framework built on
------------------------------

```

webdriverIO

mocha

chai

node.js

```

Getting started :
------------------------------

```
Make sure node is installed.

git clone 'repo-url'

cd boilerplate-webdriverio

npm install

```

Running the tests:
-------------------
```
'npx wdio' or 'npm run test'

```
